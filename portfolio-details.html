Parallel Implementation of Marathi Text News Categorisation Using GPU

Project Information

  • Category: Machine Learning
  • Project date: 03 April 2021
  • Research paper Link: Click here

The Parallel Implementation of Marathi News project focused on automating the categorization of Marathi news articles. I employed the K-NN machine learning classification algorithm and the TF-IDF term-weighting scheme to classify news into predefined categories, such as sports, economy, and entertainment. To handle the compute-intensive work, a parallel algorithm was implemented on the GPU Nvidia Tesla T4, enabling efficient processing of news articles in Marathi.
